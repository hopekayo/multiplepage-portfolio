## 🎯 十、学习检查清单

### Java 基础检查清单

```
□ 理解 Java 基本数据类型
□ 掌握面向对象编程（类、继承、接口）
□ 熟悉集合框架（List、Set、Map）
□ 会用 Stream API 处理数据
□ 理解异常处理机制
□ 掌握泛型的基本使用
```

### Spring Boot 检查清单

```
□ 能创建 Spring Boot 项目
□ 理解 IoC 和 DI
□ 会写 REST Controller
□ 掌握 @Service、@Repository 注解
□ 会配置 application.yml
□ 理解 Bean 的作用域
```

### 数据库检查清单

```
□ 会写基本的 CRUD SQL
□ 理解事务的概念
□ 掌握 JPA Repository
□ 会设计数据库表结构
□ 理解索引的作用
□ 了解数据库优化基础
```

### API 设计检查清单

```
□ 遵循 RESTful 规范
□ 统一响应格式
□ 参数验证
□ 全局异常处理
□ API 文档 (Swagger)
□ 理解 HTTP 状态码
```

### 认证授权检查清单

```
□ 理解认证与授权的区别
□ 会配置 Spring Security
□ 实现 JWT 认证
□ 理解密码加密 (BCrypt)
□ 角色权限控制
□ CORS 跨域配置
```

### DevOps 检查清单

```
□ 会用 Maven/Gradle 构建
□ 编写 Dockerfile
□ docker-compose 编排
□ GitHub Actions CI/CD
□ 云服务器部署
□ 日志查看与问题排查
```
