# 前端工程师 AI 辅助转 Java 全栈开发学习路径

> 专为有前端经验的开发者设计，借助 AI 工具高效掌握 Java 全栈开发技能

## 📚 文档结构

| 文档 | 说明 |
|------|------|
| [roadmap.md](./roadmap.md) | 详细学习路线图，包含各阶段核心知识点 |
| [resources.md](./resources.md) | 精选学习资源，包括课程、书籍、工具 |
| [weekly-plan.md](./weekly-plan.md) | 12 周详细学习计划，每周都有具体目标 |

## 🎯 适合人群

- 有 1-3 年前端开发经验
- 熟悉 HTML/CSS/JavaScript
- 了解 React/Vue 等前端框架
- 想转型全栈或后端开发

## 📋 学习路径总览

```
第 1-2 周：Java 语言基础
    ↓
第 3-4 周：Spring Boot 核心
    ↓
第 5-6 周：数据库与持久层
    ↓
第 7-8 周：API 设计与认证授权
    ↓
第 9-10 周：DevOps 与部署
    ↓
第 11-12 周：综合实战项目
```

## 🤖 AI 辅助学习策略

本学习路径特别强调 AI 工具的辅助作用：

| 场景 | AI 用法 | 示例提示词 |
|------|--------|-----------|
| 代码理解 | 解释 Java 语法、Spring 注解含义 | "解释 @Autowired 注解的作用，用前端能理解的方式" |
| 代码生成 | 生成样板代码、单元测试 | "生成一个用户注册的 Spring Boot Controller" |
| 错误调试 | 分析异常栈、提供修复方案 | "NullPointerException 可能是什么原因？" |
| 代码转换 | 将前端逻辑转换为后端实现 | "把这个 React 用户认证逻辑改成 Java Spring Security" |
| 学习答疑 | 解答概念疑惑、提供类比说明 | "用 REST API 类比解释 Spring Bean 的作用域" |

## 🚀 快速开始

1. 阅读 [roadmap.md](./roadmap.md) 了解整体学习路线
2. 根据 [weekly-plan.md](./weekly-plan.md) 制定个人学习节奏
3. 收藏 [resources.md](./resources.md) 中的资源备用
4. 开始第一周的学习！

## 💡 学习建议

- **每天至少 2 小时**专注学习
- **动手写代码**，不要只看不练
- **善用 AI 工具**，但不要依赖
- **每周复盘**，调整学习节奏
- **做项目驱动**，学完就实践

## 📅 预计完成时间

| 节奏 | 时间 | 每天投入 |
|------|------|----------|
| 快速学习 | 8-10 周 | 3-4 小时 |
| 正常节奏 | 12 周 | 2 小时 |
| 轻松学习 | 16-20 周 | 1 小时 |

## 🎓 学前知识对照表

| 前端概念 | Java 后端对应 |
|----------|--------------|
| npm/yarn | Maven/Gradle |
| package.json | pom.xml / build.gradle |
| Node.js | JVM |
| Express/Koa | Spring Boot |
| React/Vue | Thymeleaf/JSP (模板) |
| Axios | RestTemplate/WebClient |
| TypeScript | Java (类型系统) |
| ESLint | Checkstyle |
| Jest | JUnit |
| MongoDB/PostgreSQL | MySQL/PostgreSQL |

## 📦 推荐开发环境

```
- JDK 17+ (LTS 版本)
- IntelliJ IDEA (社区版免费)
- Maven 或 Gradle
- MySQL 或 PostgreSQL
- Postman (API 测试)
- Git
- Docker (部署用)
```

## 🔥 学完能做什么？

完成学习后，你将能够：

- ✅ 独立开发完整的 Spring Boot 后端应用
- ✅ 设计并实现 RESTful API
- ✅ 使用数据库存储和管理数据
- ✅ 实现用户认证和授权系统
- ✅ 部署应用到云服务器
- ✅ 借助 AI 高效解决开发问题
